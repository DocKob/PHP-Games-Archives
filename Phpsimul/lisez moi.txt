Cette version peut être installée en tant que version complète ou mise a jour
=> Pour une version complète, copiez le contenu du dossier installation sur votre FTP
=> Pour une mise à jour, supprimez tous vos fichiers, en gardant les images et le style CSS de vos templates dans un coin (Seuls ces fichiers, car les fichiers de template eux meme, ont tellement changés, qu'il est devenu impossible de récupérer les anciens). Copiez ensuite le contenu de l'archive sur votre FTP, et configurez le fichier systeme/config.php (seulement la partie "Configuration SQL"). Renommez ensuite le fichier backup_v1.3.sql en backup.dsl et le fichier backup_v1.3.php en backup.php.

Sachez qu'une autre version de la carte existe. Dans l'éditeur de menu, indiquez carte comme nom de module appelé. Ce module a été créer par Guerrier & Max485.

Pour configurer le module allopass, éditez le fichier systeme/config_allopass.php. 

Il est impossible de reprendre votre template existant dans cette version, dû au fait que tout à été modifié. Seules les images et le style pouront être récupérés.

Pour acceder a l'administration, allez à l'adresse http://votresite/admin (pareil pour les modérateurs), il vous demande le nom d'utilisateur, le mot de passe, et un code, ce code est configurable dans le fichier config.php et permet une securité supplementaire pour acceder a l'administraton, si vous etes deja connecté sur le jeu, il n'est pas utile de le taper.
Pour ajouter des admin / fondateur ou modo, vous devez vous rendre dans l'administration, et aller sur Gestion Staff, ensuite suivez les instructions.

Le code de sécurité par défaut est : phpsimul . Il est modifiable dans systeme/config.php

Lors d'une mise a jour, l'admin créé aura pour login => admin et pour pass => admin, nous vous conseillons de créer un autre compte et de supprimer celui la. Lors d'une install complete, le compte créer sera le meme que le compte utilisateur, nous vous conseillons de changer le mot de passe.
Le faite d'avoir des compte independant, vous permet de pretez votre compte joueur pour sitting... sans avoir le soucis de la moderation
