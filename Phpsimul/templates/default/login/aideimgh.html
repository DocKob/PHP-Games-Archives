Aide Avatars

Vous pouvez ajouté un avatar dans votre profil, celui ci servira pour vous representez, il apparaitra sur la carte
et a d'autres endroit selon les desirs de l'Administrateur.
Si vous decidez de ne pas mettre d'avatar, dans ce cas, un avatar par default vous sera ajouté !!


Pour obtenir un avatar par ImagesHack :

Envoyer une image sur http://www.imageshack.us/

Puis copier le code au il y a ecrit Direct link

Et mettez le en face de avatar, l'avatar vous sera donc ajouté


Pour obtenir un avatar en passant par un autre lien :

Envoyer l'image la ou vous le desirez, ou trouvez une image sur le WEB

Lorsque l'image s'affiche, faite clique droit, Propriété, et recuperer le lien de l'image
et copier le dans le champ de l'inscription prevus pour ca et l'image vous sera ajouté
